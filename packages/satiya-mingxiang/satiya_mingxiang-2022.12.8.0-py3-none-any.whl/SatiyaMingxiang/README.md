## 下载

### Docker

```
docker pull apachecn0/satiya-mingxiang
docker run -tid -p <port>:80 apachecn0/satiya-mingxiang
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install satiya-mingxiang
satiya-mingxiang <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g satiya-mingxiang
satiya-mingxiang <port>
# 访问 http://localhost:{port} 查看文档
```
