Glasses

  • id : String
    Alphanumeric ID of the glasses
  • frame_color : FrameColor
  • frame_metalness : BigDecimal
    The higher the value, the more metallic (conductive) the frame is. In real life items are either dielectric or conductive (0 or 1 metallic). Never are in between.
  • lens_color : LensColor
  • lens_reflectivity : BigDecimal
    The higher the value, the more reflective the lens are
  • lens_transparency : BigDecimal
    The higher the value, the more transparent the lens are
  • position : GlassesPosition
