IntrinsicParams

  • projection : Projection
    Currently, only perspective camera is supported
  • resolution_width : Integer
    Width resolution in pixels
  • resolution_height : Integer
    Height resolution in pixels
  • fov_horizontal : Integer
    Horizontal angle of view
  • fov_vertical : Integer
    Vertical angle of view
  • sensor_width : BigDecimal
    Horizontal size of the sensor in millimeters
  • wavelength : Wavelength
